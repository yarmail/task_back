Попробуем реализовать поиск, постраничный вывод
и сортировку на примере объектов Task

Немного теории
Для работы с постраничностью и сортировкой используются 3 объекта:
PageRequest - объект контейнер для отправки запроса
для репозитория, содержит требования постраничности и сортировки
Sort - объект-контейнер который содержит данные для сортировки.
Вкладывается в PageRequest.
Получение данных из репозитория:
Page - объект-контейнер, который содержит как результат,
так и служебные параметры (номер страницы, размер страницы и пр.)

Заводим все поля поиска, постраничность и сортировку в класс
src/main/java/task/search/TaskSearchValues.java

Далее создаем последовательно соотвествующий метод в
src/main/java/task/repo/TaskRepository.java
src/main/java/task/service/TaskService.java
src/main/java/task/controller/TaskController.java
